# Crime Data Analysis – Karnataka (Feb 2026)

## Objective
To analyze crime patterns and trends in Karnataka using Exploratory Data Analysis (EDA) techniques.

---

## Dataset
- Source: Karnataka Police Department  
- Time Period: February 2026  
- Features:
  - Heads of Crime
  - Major Heads
  - Minor Heads
  - Monthly & Yearly Crime Counts  

---

## Analysis Performed

### 1. Distribution Analysis
- Identified top crime categories using frequency analysis  
- Visualized using bar and pie charts  

### 2. Relationship Analysis
- Used pivot tables & heatmaps to analyze relationships between crime categories  
- Applied Chi-square test (p-value ≈ 0) indicating strong association  

### 3. Trend Analysis
- Month-over-Month (MoM) comparison  
- Year-over-Year (YoY) comparison  

---

## Key Insights
- Motor vehicle accidents and theft dominate crime distribution  
- Cyber crime and narcotics show contrasting trends:
  - Decline in short-term (MoM)  
  - Significant rise in long-term (YoY)  
- Crimes like cheating and molestation are increasing  
- Strong statistical association exists between crime categories  

---

## Limitations
- Dataset limited to one month  
- Trends inferred using available comparative columns  

---

## Tools Used
- Python (Pandas, NumPy)  
- Matplotlib, Seaborn  
- Scipy (Chi-square test)  
- Streamlit (Dashboard)
